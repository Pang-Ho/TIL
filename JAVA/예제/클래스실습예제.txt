교육생의 교육비에 따른 환급금을 계산하는
클래스를 작성하세요.

1. 교육생(Trainee) 클래스에는 
<필드변수>
String ==> 이름, 소속, 과정명, 
int나 double ==> 교육비, 부가금, 환급금
을 나타내는 변수를 정의합니다.

  
2. TraineeTest 클래스는 (main(String[] args) 메서드를 포함하는 실행 클래스) 
작성합니다.
 2-1> 명령행 매개변수를 이용하여 사용자로부터
   이름, 소속, 과정명, 교육비, 부가금을 입력받습니다.

 실행 : 
run-run confirations-arguments-program arguments
이자바 멀티캠퍼스 JSP프로그래밍 50000 5000



2-2> 교육생 클래스의 객체를 생성하여 stu
 라는 이름의 변수로 참조하도록 합니다.

2-3> 2-1 입력 5개 변수를 이름, 소속, 과정명,
교육비, 부가금 변수값으로 초기화합니다.
 예) stu.이름변수 = args[0];


2-4>  환금급을 계산하여 교육생 클래스의 환급금 변수에 저장합니다.

  - 환급금은 
   과정명이 "자바프로그래밍" 이면
   입력한 교육비의 25% + 부가금 이고,
   
   과정명이 "JDBC프로그래밍" 이면
   입력한 교육비의 20% + 부가금 이고,
   
   과정명이 "JSP프로그래밍" 이면
   입력한 교육비의 15% + 부가금 으로        
   계산하여 초기화합니다.

   그밖의 과정명은 환급금은 0 입니다.


2-5>  환급금은 0 이 아니면 
stu 내의 이름, 소속, 과정명, 교육비, 부가금, 환급금을 출력합니다. 
환급금이 0 이면 
"잘못 입력되었습니다"는 메시지를 출력합니다.

<입출력의 예1>
run-run confirations-arguments-program arguments

이름=이자바
소속=멀티캠퍼스
과목명=JSP프로그래밍

환급금=12500원

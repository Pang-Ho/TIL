# 재귀용법

* 함수 안에서 동일한 함수를 호출하는 형태

## 팩토리얼 구현

```java
class fac {
    public int factorial1(int num) {
        int result = 1;
        for(int i = 2 ; i < num + 1 ; i ++) {
            result = result * i;
        }
        return result;
    }
    public int factorial2(int num) {
        
        int result = 1;
        if(num > 1) {
	        result = num * factorial2(num - 1);            
        }
		
        return result;
    }
}
```

* str 문자열에 char s가 몇개 들어있는지 확인하는 방법

```java
class CountChar {
    public int count (String str, char s) {
        int num = 0;
        for(int i = 0 ; i < str.length() ; i++) {
            if(str.charAt(i) == s) {
                num++;
            }
        }
        return num;
}
```
